# Jogo da Forca - PHP

Projeto desenvolvido para a disciplina de Programação.

## Descrição

Este projeto consiste em uma implementação do jogo da forca utilizando PHP, HTML e CSS, sem uso de banco de dados.

O jogador deve tentar descobrir a palavra secreta digitando letras até acertar a palavra completa ou atingir o limite de erros.

## Tecnologias Utilizadas

- PHP
- HTML
- CSS
- Git
- GitHub

## Funcionalidades

- Exibição da palavra oculta
- Entrada de letras pelo usuário
- Controle de letras digitadas
- Verificação de acertos e erros
- Mensagem de vitória
- Mensagem de derrota
